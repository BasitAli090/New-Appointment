# Icon Setup Guide

## Required Icons

### For Windows EXE:
- **File**: `assets/icon.ico`
- **Size**: 256x256 pixels (or larger, will be resized)
- **Format**: ICO format

### For Android APK:
- **Files**: Multiple sizes (Android Studio will generate these automatically)
- **Recommended**: `assets/icon.png` (1024x1024 pixels)
- **Format**: PNG format

## Creating Icons

### Option 1: Online Tools
1. Create your icon image (1024x1024 PNG)
2. Convert to ICO: https://convertio.co/png-ico/ or https://icoconvert.com/
3. Place files:
   - `assets/icon.ico` (for Windows)
   - `assets/icon.png` (for Android - will be auto-resized)

### Option 2: Using Image Editing Software
- Use Photoshop, GIMP, or any image editor
- Create 1024x1024 PNG
- Export as ICO for Windows
- Export as PNG for Android

### Option 3: Quick Placeholder
If you don't have icons yet, the apps will use default icons. You can add custom icons later.

## Icon Design Tips
- Use simple, recognizable designs
- Ensure icons are clear at small sizes
- Use high contrast colors
- Test icons on different backgrounds
